# Explorer-Robot
Build your explorer robot! It is able to find sounds, gas sources and ligth points!

## Getting Started

First of all it's required a rover! You can find the instruction to build one int the "project" folder.
After that you can upload the skect on the board, tune sume parametres and then have fun!

### Prerequisites

You need to install [LM298](https://github.com/AndreaLombardo/L298N) library by Andrea Lombardo
